results_table_by_class_conf_bs3

Using a conf class and a set of data, automatically populate and return a results table.

This is the Bootstrap 3 version of this display.  It affixes itself directly to the DOM using jQuery at elt_id.

Parameters

class_confa bbop.golr.conf_class
golr_respa bbop.golr.response
linkera linker object; see <amigo.linker> for more details
handlera handler object; see <amigo.handler> for more details
elt_idthe element id to attach it to
selectable_p[optional] whether to create checkboxes (default true)

this object

process_entry

bbop.widget.display.results_table_by_class_conf_b3.prototype.process_entry = function(
   bit,
   field_id,
   document,
   display_context
)

The function used to render a single entry in a cell in the results table.  It can be overridden to specify special behaviour.  There may be multiple entries within a cell, but they will all have this function individually run over them.

This function can access this._golr_response (a bbop.golr.response), this._linker (a bbop.linker), and this._handler (a bbop.handler).

Arguments

bitstring (?) for the one entry in the cell
field_idstring for the field under consideration
documentthe single document for this item from the solr response

Returns

string or empty string (‘’)
